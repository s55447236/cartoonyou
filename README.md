# CartoonYou

CartoonYou 是一个基于 AI 技术的照片卡通化服务，能够将您的照片转换成独特的卡通风格头像。

## 功能特点

- 多种卡通风格选择
- 实时预览效果
- 支持表情动画
- 快速生成和下载
- 用户友好的界面

## 技术栈

- HTML5
- CSS3
- JavaScript
- 响应式设计

## 项目结构

```
cartoonme/
├── images/          # 图片资源
├── styles/          # CSS样式文件
├── scripts/         # JavaScript文件
├── index.html      # 主页
├── about.html      # 关于页面
├── privacy.html    # 隐私政策
└── terms.html      # 服务条款
```

## 开始使用

1. 克隆仓库

```bash
git clone https://github.com/yourusername/cartoonme.git
```

2. 打开项目

```bash
cd cartoonme
```

3. 在浏览器中打开 `index.html` 文件
